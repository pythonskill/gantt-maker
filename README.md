# Gantt Maker

Проект для создания диаграмм Ганта с использованием ReactJS на фронтенде и Quarkus на бэкенде.

## Структура проекта

- `frontend/` - ReactJS приложение для построения диаграммы Ганта
- `backend/` - Quarkus backend для обработки данных

## Запуск

### Frontend
```bash
cd frontend
npm start
```

### Backend
```bash
cd backend
./mvnw quarkus:dev
```
